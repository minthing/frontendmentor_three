* 처음에는 그냥 `<div>`로 묶고 `<strong>`을 사용했는데, 각 box가 `<ㅗ>`과 `<p>`를 가진 `<article>`로 표현될 수 있다고 피드백 해 줌. html5에서 신규로 등장한 요소는 어쩐지 잘 사용하지 않게 되었는데 어차피 공부를 위한 프로젝트이니 이제 이 방식도 고려해서 더 좋은 게 있으면 차용하도록 하자.
